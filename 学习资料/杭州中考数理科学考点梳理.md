# 杭州中考（2026）数学·科学 高频考点梳理
**整理时间：2026-03-21 | 适用：初一至初三 | 总分：数学120 + 科学160**

---

## 📐 数学（120分）

### 一、高频考点分布

| 模块 | 分值占比 | 难度 | 备注 |
|------|----------|------|------|
| 函数（一次/反比例/二次） | ~30% | ★★★ | 压轴题常客 |
| 几何证明（全等/相似/圆） | ~25% | ★★★ | 辅助线是难点 |
| 方程与不等式 | ~20% | ★★ | 基础必须拿满 |
| 数与式（实数/代数式） | ~15% | ★★ | 选择填空高频 |
| 综合应用 | ~10% | ★★★ | 压轴题 |

### 二、各年级重点（窝窝现处初一，承上启下）

#### 七年级（上）
- 有理数、实数运算
- 代数式与整式
- 一元一次方程
- 几何初步（线段、角）

#### 七年级（下）
- 二元一次方程组
- 不等式与不等式组
- 平面直角坐标系
- 平行线与相交线

#### 八年级
- 全等三角形
- 一次函数
- 整式乘除与因式分解
- 分式

#### 九年级
- 二次函数
- 圆
- 相似三角形
- 反比例函数
- 三角函数

### 三、压轴题类型（浙江特色）

1. **函数与几何综合** — 建立坐标系，研究图形性质
2. **几何证明（新定义/创新）** — 课本题原型+辅助线
3. **实际应用题** — 用方程/函数解建模问题

### 四、易错点（浙江考生普遍丢分区）

- 几何证明跳步（缺少依据）
- 函数图像平移方向搞反
- 分式方程忘记检验
- 二次函数最值讨论不全

### 五、答题模板（中考必备）

**几何证明题：**
```
1. 读题→标已知条件
2. 画草图→标注关键长度/角度
3. 分析思路（从结论反推，或从已知正向推）
4. 规范书写：∵（因为）∴（所以）
```

**函数应用题：**
```
1. 设未知数
2. 建立等量关系
3. 解方程/不等式
4. 检验（是否符合实际意义）
5. 答
```

## 🧪 科学（160分）

> 科学包含物理、化学、生物、地理，综合性强

### 一、高频考点分布（按学科）

| 学科 | 主要内容 | 分值占比 |
|------|----------|----------|
| 物理 | 力学、电学、热学、光学 | ~45% |
| 化学 | 酸碱盐、化学方程式、实验 | ~30% |
| 生物 | 细胞、遗传变异、生态 | ~15% |
| 地理 | 地球与地图、气候 | ~10% |

### 二、物理高频考点

#### 力学（八年级重点）
- **力和运动**：重力、弹力、摩擦力；二力平衡；运动和力
- **压强**：固体压强、液体压强、大气压强、流体压强
- **浮力**：阿基米德原理；物体的浮沉条件
- **功和机械能**：功、功率、机械效率

#### 电学（九年级重点）
- **电路分析**：串并联识别、电流/电压/电阻规律
- **欧姆定律**：I=U/R；动态电路分析
- **电功率**：P=UI；电能和电热
- **家庭电路**：安全用电

#### 热学
- 物态变化（熔化、凝固、汽化、液化）
- 内能和热机

#### 光学
- 光的反射与折射
- 凸透镜成像规律

### 三、化学高频考点（九年级）

#### 酸碱盐（最重要！）
- 酸的通性（与金属、金属氧化物、碱、盐反应）
- 碱的通性
- 中和反应
- 常见盐的性质（碳酸钙、氯化钠、硫酸铜）
- 复分解反应条件

#### 化学计算
- 化学方程式书写与计算
- 溶质质量分数

#### 实验操作
- 常见仪器识别
- 基本操作规范（过滤、蒸发、稀释）
- 氧气/二氧化碳的制取

### 四、生物高频考点

- 细胞结构（植物/动物细胞对比）
- 光合作用与呼吸作用
- 遗传与变异（DNA、基因）
- 人体系统（消化、循环、呼吸）
- 传染病与免疫

### 五、科学备考建议

1. **物理**：理解概念 > 死记公式 > 大量刷题
2. **化学**：方程式是基础，实验是灵魂
3. **生物**：图文结合，构建知识网络
4. **总策略**：七年级打基础，八年级攻力学/电学，九年级综合冲刺
